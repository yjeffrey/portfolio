The Public Piano

This is a public piano. Anybody can hear anything that is played by anybody else in real-time. You could open this page in two windows or tabs, and playing notes in one page would cause the notes to be played in the other page.

This demo primarily uses socket.io and web audio api, which may or may not be supported by the browser you are using.

Socket.io is used to transmit notes played, and other interactions between clients quickly in real-time. If you're using a relatively modern browser, web-sockets should be the underlining transport being used. This is the preferred transport as it uses persistent connections instead of other hacky socket implementations such as long-polling.

There was also a large amount of work put in to ensure that this part of the application is scalable. The number of players shown is accurate when several web-app instances are used. The number is also correct when an instance fails unexpectedly, and when an instance is restarted.

Click anywhere inside the green box to start playing!
